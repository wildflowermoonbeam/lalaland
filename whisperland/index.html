whisperland

whisperland is a pinterest-inspired role playing game. gameplay takes about 10-15 minutes.

system requirements:

  • for the best experience, please navigate to whisperland on a desktop/laptop computer
  • enable javascript
  • expand your web browser to full screen

press the button below to get started.

▶ start ◀
note: the game will occasionally ask you to provide text input. any information you submit is used only in-game; it is not saved or submitted anywhere.
